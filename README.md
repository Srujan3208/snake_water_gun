# Snake Water Gun Game

## Project Overview
This repository contains a Python-based implementation of the classic **Snake–Water–Gun** game.  
The game allows a user to play against the computer and provides **voice feedback** for the game result using the `pyttsx3` text-to-speech library.

## Features
- Command-line based gameplay
- User vs Computer mode
- Input validation for incorrect entries
- Voice output announcing the game result
- Simple and beginner-friendly Python logic

## Technologies Used
- Python 3
- pyttsx3 (Text-to-Speech)

## Installation and Setup
1. Ensure Python 3 is installed on your system.
2. Install the required dependency using pip:
```bash
pip install pyttsx3

## How to Run the Program
Run the following command in the project directory:
```bash
python snake_water_gun.py

---
## Usage Instructions
- Enter `s` for Snake  
- Enter `w` for Water  
- Enter `g` for Gun  

The computer will randomly select its choice, and the result will be displayed along with voice feedback.
